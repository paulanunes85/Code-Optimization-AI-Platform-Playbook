# Prerequisites

Before getting started with the Code AI Playbook, ensure you have the following prerequisites in place:

## Account Requirements

- A GitHub account with appropriate access (Enterprise, Business, or Professional plan)
- Administrative permissions for your organization or repository
- Azure subscription with appropriate permissions (for Azure AI Foundry resources)

## Development Environment

- One of the following IDEs with the latest updates:
  - Visual Studio 2025
  - Visual Studio Code
  - JetBrains IDE (IntelliJ IDEA, PyCharm, Rider, etc.)

## Required Tools

- GitHub CLI installed (version 2.40.0 or later)
- Git (version 2.40.0 or later)
- Node.js (version 18.0.0 or later) for web-based projects
- .NET SDK (version 8.0 or later) for .NET projects
- JDK (version 17 or later) for Java projects
- Python (version 3.10 or later) for Python projects

## Network Requirements

- Stable internet connection
- Access to GitHub.com and Azure services
- If behind a corporate firewall, ensure the following domains are allowlisted:
  - github.com
  - api.github.com
  - copilot.github.com
  - azure.microsoft.com
  - *.azurewebsites.net
  - *.foundry.azure.com

## Security Considerations

- Ensure your organization's security policies allow for AI-assisted code generation
- Review and understand GitHub Copilot's data usage policies
- Consider setting up private repositories for sensitive code
- Implement appropriate code review processes for AI-generated code

## Knowledge Prerequisites

- Basic understanding of GitHub workflows and actions
- Familiarity with CI/CD concepts
- Understanding of your codebase and development processes
- Basic knowledge of Azure services (for Azure AI Foundry integration)

## Next Steps

Once you have all prerequisites in place, proceed to the [Setup](setup) section to configure GitHub Copilot Agent and Azure AI Foundry for your development environment.
